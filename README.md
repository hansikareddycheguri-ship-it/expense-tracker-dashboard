# Personal Expense Tracker Dashboard

A responsive frontend dashboard UI built using HTML and CSS.

## Features

* Expense summary cards
* Expense categories section
* Recent transactions section
* Flexbox-based layout
* Modern card UI design
* Hover effects and responsive spacing

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Font Awesome Icons
